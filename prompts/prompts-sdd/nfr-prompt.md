# PROMPT: Non-Functional Requirements (NFR)

Usando los FR definidos y la visión general del sistema:

1. Genera todos los NFR necesarios para un sistema profesional.
2. Incluye categorías:
   - Rendimiento
   - Fiabilidad
   - Seguridad
   - Usabilidad
   - Mantenibilidad
   - Escalabilidad
3. No inventes features; solo restricciones de calidad.

Formato:

## Non-Functional Requirements
- **NFR-01 (Seguridad):** …
- **NFR-02 (Rendimiento):** …
…

🚀 CÓMO USAR

Opción 1 - Script (30 segundos):
$ ./run.sh                    # Linux/Mac
$ run.bat                     # Windows

Opción 2 - Docker (1 minuto):
$ docker-compose up -d

Opción 3 - Manual:
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python3 -m uvicorn main:app --reload

- API: http://localhost:8000
- Docs: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

🔐 REGLAS CRÍTICAS IMPLEMENTADAS

⚠️  RN-11: Stock nunca puede ser negativo
    → Validación en registrar_salida()
    → Garantiza no deuda físicamente

⚠️  RN-14: Stock actualizado inmediatamente
    → Transacción atómica en movimientos
    → Sin estados intermedios inconsistentes

⚠️  RN-26: Soft delete obligatorio
    → Nunca se ejecuta SQL DELETE
    → DELETE endpoint setea activo=false
    → Preserva integridad histórica
